## Description

This project runs on [Nest](https://github.com/nestjs/nest). To use it, just follow these instructions   above.

## Installation

```bash
$ git clone https://github.com/allanmelo/smartranking.git
$ cd smartranking
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
